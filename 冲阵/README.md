# 冲阵 - 游戏系统文档

## 游戏概述

**冲阵** 是一款融合 MOBA 分路策略、皇室战争资源管理和塔防自动战斗的轻量级对战游戏。

### 核心玩法流程

```
出身选择 → 球形世界地图 → 格子战斗 → 占领推进 → 挑战中央宝藏
```

---

## 一、出身选择系统

### 1.1 四种出身

| 出身 | 名称 | 副标题 | 对应场景 | 图标 |
|------|------|--------|----------|------|
| 平原 | 山丘领主 | 高地守护者 | 陆地 (land) | ⛰️ |
| 海底 | 深海行者 | 深渊探索者 | 海底 (ocean) | 🌊 |
| 太空 | 星际旅者 | 虚空漫步者 | 太空 (space) | 🚀 |
| 草原 | 草原之魂 | 风之骑手 | 草原 (grassland) | 🌿 |

### 1.2 选择界面
- 四个出身选项以卡片形式展示
- 悬停时卡片放大并显示发光边框
- 点击后进入世界地图

## 二、球形世界地图系统

### 2.1 地图结构

```
                    [敌方出生点]
                         ↑
         [敌方出生点] ← 中央宝藏 → [敌方出生点]
                         ↓
                    [玩家出生点]
```

- **5层同心圆环**：内圈8格，每外圈多4格
- **中央宝藏**：金色星形，带脉冲光环
- **四个出生点**：位于球形边缘四个方向

### 2.2 格子系统

| 环数 | 格子数 | 怪物等级 | 难度 |
|------|--------|----------|------|
| 1环 | 8格 | Lv1 | 最简单 |
| 2环 | 12格 | Lv2 | 简单 |
| 3环 | 16格 | Lv3 | 中等 |
| 4环 | 20格 | Lv4 | 困难 |
| 5环 | 24格 | Lv5 | 最困难 |

### 2.3 迷雾系统
- 玩家只能看到周围1环范围的格子
- 其他格子显示为迷雾状态
- 远处出生点在迷雾中隐约可见（带光晕效果）

### 2.4 战斗机制
- 每个未占领的格子都有怪物
- 点击怪物格子进入战斗
- 胜利后格子显示 ✓ 标记
- 占领内圈所有格子后可挑战中央宝藏

## 三、战斗系统

### 3.1 战场布局

```
┌─────────────────────────────────────┐
│           敌方基地 (1000HP)          │
├─────────────────────────────────────┤
│                                     │
│         弧形道路（左/中/右）          │
│                                     │
├─────────────────────────────────────┤
│     英雄站位区（每路一个英雄）         │
├─────────────────────────────────────┤
│     士兵屯兵区（显示屯兵数量）         │
├─────────────────────────────────────┤
│           我方基地 (1000HP)          │
└─────────────────────────────────────┘
```

### 3.2 兵种系统

| 兵种 | 生命值 | 攻击力 | 攻击速度 | 攻击范围 | 特点 |
|------|--------|--------|----------|----------|------|
| 近战 (melee) | 150 | 15 | 1.0 | 25 | 高生命，近战 |
| 远程 (ranged) | 80 | 12 | 1.5 | 80 | 低生命，远程 |
| 坦克 (tank) | 300 | 8 | 0.5 | 25 | 极高生命，肉盾 |

### 3.3 出兵机制
- **屯兵系统**：每路有独立的屯兵区
- **出兵消耗**：出兵时消耗屯兵数量
- **屯兵增长**：每2秒自动增加1个屯兵
- **快捷键**：Q(左路) / W(中路) / E(右路)

### 3.4 军团战斗系统

#### 战斗触发
- 小兵遇到敌人自动停止移动
- 周围友军（80像素范围内）自动加入战斗
- 最多5人组成军团

#### 阵型系统
- 自动排列成弧形包围敌人
- 军团成员保持阵型偏移位置

#### 协同攻击
- 军团成员同时攻击同一目标
- 目标死亡后自动切换新目标
- 目标脱离范围后退出战斗

#### 视觉效果
- 军团光环：显示军团人数的脉冲光环
- 战斗指示：连接战斗双方的虚线和⚔标记
- 战斗状态：单位周围显示橙色光晕

## 四、英雄技能系统

### 4.1 英雄配置

| 位置 | 英雄 | 技能名称 | 伤害 | 冷却 | 特效 |

| 中路 | 英雄 | 烈焰风暴 | 150 | 10秒 | 火焰爆炸 |
| 右路 | 英雄 | 圣光审判 | 120 | 9秒 | 金色光柱 |

### 4.2 自动释放机制
- 自动寻找敌人最多的区域
- 技能范围：80像素
- 对范围内所有敌人造成伤害

### 4.3 技能特效

#### 雷霆一击
- 蓝色闪电圈扩散
- 6道闪电射线
- 金色伤害数字

#### 烈焰风暴
- 火焰圈扩散
- 8个火焰粒子环绕
- 橙色伤害数字

#### 圣光审判
- 金色光柱从天而降
- 12道圣光射线
- 中心光球爆发

## 七、技术架构

### 7.1 文件结构

```
冲阵/
├── index.html              # 主页面
├── js/
│   ├── main.js            # 游戏入口
│   ├── core/
│   │   ├── Game.js        # 游戏主类
│   │   ├── GameFlow.js    # 游戏流程管理
│   │   ├── Entity.js      # 实体基类
│   │   └── Scene.js       # 场景管理
│   ├── systems/
│   │   ├── MoveSystem.js      # 移动系统
│   │   ├── CollisionSystem.js # 碰撞系统
│   │   └── RenderSystem.js    # 渲染系统
│   ├── entities/
│   │   └── Unit.js        # 单位工厂
│   └── utils/
│       ├── Vector2.js     # 向量工具
│       └── Pool.js        # 对象池
```

### 7.2 核心类说明

#### Game 类
- 管理游戏循环和全局状态
- 处理英雄技能系统
- 管理基地血量

#### GameFlow 类
- 管理出身选择 → 世界地图 → 战斗 的流程
- 处理用户输入事件
- 协调不同阶段切换

#### WorldMap 类
- 球形地图生成和渲染
- 格子状态管理（占领/未占领）
- 怪物生成和战斗触发

#### Entity 类
- 实体属性和状态
- 战斗状态管理
- 伤害数字和攻击特效

### 7.3 系统架构

```
GameFlow (流程管理)
    ├── WorldMap (世界地图)
    └── Game (战斗游戏)
            ├── MoveSystem (移动)
            ├── CollisionSystem (碰撞/战斗)
            └── RenderSystem (渲染)
```

## 八、操作指南

### 8.1 出身选择阶段
- 点击四个出身卡片之一选择

### 8.2 世界地图阶段
- 鼠标悬停查看格子信息
- 点击可见格子上的怪物进入战斗

### 8.3 战斗阶段
- **Q/W/E**：左路/中路/右路出兵
- **点击按钮**：对应路出兵
- **场景切换**：点击右上角图标
- **路数切换**：点击右侧数字按钮
